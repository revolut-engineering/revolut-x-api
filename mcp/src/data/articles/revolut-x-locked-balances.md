# Why Is Part of My Revolut X Portfolio Balance Unavailable?

## Locked Balances

If you have any pending limit, TP/SL, conditional, or TWAP orders, the amount that you used to instruct the order will be locked to ensure that you have sufficient balance when execution is triggered.

In case you need to use the balance for other purposes, you may cancel the pending orders before execution is triggered.

## To Cancel a Limit, Conditional, TWAP, or TP/SL Order:

1. Go to **'Trade'**
2. Go to the **'Activity'** widget
3. Select the order under **'Orders'**
4. Tap **'X'** on the right side of the pending order you want to cancel
