# Failed Orders in Revolut X

The order can be cancelled or rejected by the system after it has been placed.

## Common Reasons of Market Order Failures

- **Insufficient liquidity**
- **Slippage protection** (the estimated order price is too far from the mid rate in the order book)
- **Self-matching protection** (order is matched against one of the customer orders)

## Common Reasons of Limit Order Failures

- **Post-only order can't get executed as a taker**
- **The order has expired**
