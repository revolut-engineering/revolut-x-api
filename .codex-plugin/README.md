# revolut-x

OpenAI Codex plugin for [Revolut X](https://exchange.revolut.com/) — a crypto exchange. Adds 7 skills that teach Codex to use the `revx` CLI for auth setup, market data, account queries, order placement, live monitoring, and grid trading.

## What's included

Seven skills covering the full Revolut X CLI surface:

- **revx-auth** — install the CLI, generate an Ed25519 keypair, register the API key; auto-resolves auth errors thrown by other skills
- **revx-market** — currencies, trading pairs, tickers, candles, order book depth
- **revx-account** — balances, open orders, order history, fills, private/public trade history
- **revx-trading** — place market and limit orders, cancel orders (always confirms before any real-money action)
- **revx-monitor** — live alerts on price thresholds, RSI, EMA crossovers, MACD, Bollinger bands, volume spikes, spread, OBI, ATR breakouts (long-running background process)
- **revx-telegram** — set up Telegram bot connectors so monitor and grid bot alerts arrive as messages
- **revx-strategy** — backtest, optimize, and run grid trading strategies (live mode requires explicit user confirmation)

## Requirements

The skills invoke the `revx` CLI, which needs:

- Node.js ≥ 20
- A Revolut X account with an API key registered against an Ed25519 public key generated by `revx configure`

Install the CLI:

```bash
npm install -g @revolut/revolut-x-cli
revx configure
```

The first time Codex needs an authenticated endpoint, the **revx-auth** skill walks through key generation and registration. Only the steps that require the user (pasting the API key, registering the public key in their Revolut X profile) are surfaced; everything else is auto-applied.
